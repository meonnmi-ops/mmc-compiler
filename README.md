# MMC (Myanmar Code) Transpiler v8.1

**ကမ္ဘာ့ပထမ မြန်မာဘာသာစကား ပရောဂျက်ဖွင့်ဆောင်ရေးစက်**

MMC (Myanmar Code) ဆိုသည်မှာ မြန်မာဘာသာစကားဖြင့် ကုဒ်ရေးသား၍ Python ဘာသာစကားသို့ အလိုလိုပြောင်းလဲပေးသော ပရောဂျက်ဖွင့်ဆောင်ရေးစက် (transpiler) ဖြစ်သည်။ v8.1 တွင် စကားလုံးစုံ (keyword) ၁၈၁ ခု ပါဝင်ပြီး OOP, Dict, Error Handling, File I/O, Async, Decorator, Type Hint စသည့် ဖွင့်ဆောင်မှုများ အကြောင်း အပြည့်အဆုံး ထောင့်ပါသည်။

## ✨ ထူးခြားချက်များ

- **စကားလုံး ၁၈၁ ခု** — ကုဒ်ရေးသားသည့်တိုင်း မြန်မာစကားကိုသာ အသုံးပြုနိုင်ပြီး Python လိုပါ မကြားပါဘဲ သို့မဟုတ်ရေးနိုင်သည်
- **OOP (Object-Oriented Programming) ထောင့်ပါ** — Class, Inheritance, Super, isinstance စသည့် အခုံအရေးများကို ပေါင်းရမ်းပေးသည်
- **File I/O** — ဖိုင်ဖွင့်ခြင်း၊ ဖတ်ခြင်း၊ ရေးခြင်းတို့ကို မြန်မာစာလုံးဖြင့်သာ ဆောင်ရွက်နိုင်သည်
- **Error Handling** — `ကြိုးစား` / `ဖမ်းမိ` / `နောက်ဆုံး` (try/except/finally) ဖြင့် ပွဲဖွင့်များကို ကိုင်ရှင်စွာ ထိန်းသိမ်းနိုင်သည်
- **Type Hints** — ဆန်ခြေမျှဝေါ်ချက်များ (type annotations) ပေးနိုင်ပြီး ကုဒ်ကို ပိုမို ကြီးကြပ်စွာ ရေးနိုင်သည်
- **Async/Await** — `မတူညီ` / `မျှော်` ဖြင့် asynchronous programming ကိုလည်း ပေါင်းရမ်းထားသည်
- **Termux မှ အလုပ်လုပ်နိုင်ပါသည်** — Android စက်ပစ္စည်းပေါ်တွင် Python 3.8+ ဖြင့် အခုပါ လုပ်ဆောင်နိုင်သည်

## 📦 v8.1 တွင် အသစ်ထည့်သွင်းလုပ်ထားသောအတိုင်း

- `စာရင်း` — List Comprehension ထောင့်ပါ
- Decorator Syntax ထောင့်ပါ (`@ decorator`)
- `လံဘ်ဒါ` — Lambda ဖော်နည်း ထောင့်ပါ
- `မှန်လျှင်` — Inline If/Else (Ternary) ထောင့်ပါ
- `အတူ` — With Statement (Context Manager) ထောင့်ပါ

## 🚀 အသုံးပြုပုံ

### ၁။ Repository ကို clone လုပ်ပါ

```bash
git clone https://github.com/meonnmi-ops/mmc-compiler.git
cd mmc-compiler
```

### ၂။ Termux တွင် အသစ်တင်သွင်းရန်

```bash
pkg install python git
git clone https://github.com/meonnmi-ops/mmc-compiler.git
cd mmc-compiler
```

### ၃။ MMC ကုဒ်ကို Python သို့ ပြောင်းပါ

```bash
python3 mmc_transpiler.py examples/hello.mmc
```

### ၄။ စာသားလုံးချင်းပါ ရေးပြီး ပြောင်းနိုင်သည်

```bash
python3 mmc_transpiler.py -c 'ပုံနှိပ်("မကြီးမားသော ကမ္ဘာ")'
```

### ၅။ ပြောင်းထားသော Python ကုဒ်ကို လုပ်ဆောင်ပါ

```bash
python3 mmc_transpiler.py examples/hello.mmc | python3
```

## 📖 ကုဒ်ရေးနည်း ဥပမာများ

### Hello World

```mmc
ပုံနှိပ်("မကြီးမားသော ကမ္ဘာ")
```

### ကိန်း သတ်မှတ်ခြင်း

```mmc
ကိန်း အမျိုး = ၅
ကိန်း လက်ရှိ = "MMC v8.1"
ပုံနှိပ်(အမျိုး, လက်ရှိ)
```

### အတွက် လမ်းကြောင်း (For Loop)

```mmc
အတွက် ကွက် ထဲတွင် ပတ်လမ်း(၅):
    ပုံနှိပ်(ကွက်)
```

### အကယ်၍ / မဟုတ်ပါက (If / Else)

```mmc
ကိန်း တန်ဖိုး = ၉၀

အကယ်၍ တန်ဖိုး >= ၈၀:
    ပုံနှိပ်("အရေးကြီးပါသည်")
မဟုတ်လျှင် တန်ဖိုး > ၅၀:
    ပုံနှိပ်("အသေးစိတ် အရေးကြီးပါသည်")
မဟုတ်ပါက:
    ပုံနှိပ်("ထပ်လုပ်ရန် လိုပါသည်")
```

### အတန်း (Class - OOP)

```mmc
အတန်း ခေါ်ဆိုချက်:
    အဓိပ္ပာယ် စနစ်(ခွဲ, အမည်):
        ခွဲ.ခွဲ = ခွဲ
        ခွဲ.အမည် = အမည်

    အဓိပ္ပာယ် ကြက်ခြေ():
        ပြန်ပေး f"{ခွဲ.အမည်} ၏ ခွဲမှာ {ခွဲ.ခွဲ} ရှိသည်"

ကိန်း ပရောဂျက် = ခေါ်ဆိုချက်("MMC", "မြန်မာ Code")
ပုံနှိပ်(ပရောဂျက်.ကြက်ခြေ())
```

### ဖိုင် ဖွင့်ခြင်း / ဖတ်ခြင်း / ရေးခြင်း (File I/O)

```mmc
အတူ ဖိုင် = ဖွင့်("data.txt", "w"):
    ဖိုင်.ဖိုင်ရေး("Hello MMC")

ကိန်း အချက် = ဖွင့်("data.txt").ဖိုင်ဖတ်()
ပုံနှိပ်(အချက်)
```

### ပွဲဖွင့် ထိန်းသိမ်းခြင်း (Try / Except)

```mmc
ကြိုးစား:
    ကိန်း အမျိုး = ၁၀ / ၀
ဖမ်းမိ ပွဲဖွင့် ဖြစ်ပွဲ:
    ပုံနှိပ်("ပွဲဖွင့်ဖြစ်သည်: ", ဖြစ်ပွဲ)
```

### စာရင်း Comprehension (v8.1 အသစ်)

```mmc
ကိန်း ကွက်တမ်း = [x * x အတွက် x ထဲတွင် ပတ်လမ်း(၁၀)]
ပုံနှိပ်("ကွက်တမ်း: ", ကွက်တမ်း)
```

## 🔑 စကားလုံးစုံ အုတ် (Keyword Categories)

| အမျိုးအစား | ဥပမာ စကားလုံးများ | Python ပြောင်းချက် |
|---|---|---|
| Statement | `အကယ်၍` `အတွက်` `ပြန်ပေး` | `if` `for` `return` |
| Function | `ပုံနှိပ်` `အရှည်` `ပတ်လမ်း` | `print` `len` `range` |
| Method | `ထည့်သွင်း` `ဖိုင်ဖတ်` `စာခွဲ` | `append` `read` `split` |
| Value | `မှန်` `မှား` `ဘာမှမရှိ` | `True` `False` `None` |
| Type | `အတည်` `ဒစ်` `စာသား` | `int` `float` `str` |
