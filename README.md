# 🇲🇲 Native MMC AI Compiler

LLVM-based compiler for Myanmar Programming Language (MMC) that generates native ARM64 binaries with AI inference capabilities.

## Features
- ✅ Native ARM64 ELF binary generation
- ✅ LLVM 18.x based IR generation
- ✅ GGML AI inference integration
- ✅ Termux/Android compatible
- ✅ No Python runtime required

## Quick Start
```bash
./bin/mmc -o hello examples/hello.mmc
./hello
```
